# 翻译字幕校对规则

## 1. 基本任务目标

基于原始字幕（主要为英语），对现有翻译字幕（主要为中文）进行校对与修正，最终输出时间轴准确、译文自然、无明显漏译/错译的翻译字幕文件。

---

## 2. 时间轴规则

1. **以原始字幕幕时间轴为基准**
   - 逐条根据原始字幕时间轴重新整理翻译字幕。
   - 翻译字幕应尽量对应原始字幕的起止时间。

2. **逐段校对时间轴**
   - 后续字幕翻译/校对任务中，采用：
     > 逐段翻译/校对 → 立即核对该段时间轴 → 再继续下一段
   - 不采用全文完成后才统一校对的方式。

3. **避免使用 Python 批处理时间轴**
   - 为避免时间轴错乱，不使用 Python 对字幕进行自动批量处理。
   - 应人工逐段检查时间轴、文本对应关系和上下文。

4. **最终整体复核**
   - 完成全部译文后，需要再次整体比对原始原始字幕与翻译字幕。
   - 重点检查：
     - 是否有漏译
     - 是否有错位
     - 是否出现仅有英文、仅有中文的异常段落
     - 是否有译文字数与原文长度比例异常的条目

---

## 3. 漏译处理规则

1. **普通对白漏译必须补充**
   - 如果原始字幕中有内容，而翻译字幕缺失，应补译。

2. **明显剧情信息漏译必须补充**
   - 包括人物行动、关键台词、剧情转折、重要名词等。

3. **SDH 部分缺译可忽略**
   - SDH 指为听障观众提供的说明性字幕，例如：
     - `[door opens]`
     - `[music playing]`
     - `[sighs]`
     - `[explosion]`
   - 若这类内容在翻译字幕中缺失，可以不强制补译。

4. **歌词需要正常翻译**
   - 如果原始字幕中出现歌词，不按 SDH 忽略。
   - 应根据语境正常翻译。

---

## 4. 错译处理规则

1. **明显错译必须修正**
   - 包括：
     - 人物关系译错
     - 动作、事件译错
     - 否定/肯定译反
     - 时间、地点、数字译错
     - 科技术语、职业术语、剧情设定译错

2. **轻微表达不自然可以润色**
   - 在不改变原意的前提下，让翻译字幕更自然、更符合中文观影习惯。

3. **保留原片风格**
   - 译文应尽量贴合影片类型、人物性格和时代背景。
   - 如用户提供 IMDB 编号，应先查询影片/剧集信息，再进行翻译或校对。

---

## 5. 多余翻译字幕处理规则

1. **如果翻译字幕条目多于原始字幕，原则上保持不变**
   - 不随意删除翻译字幕中额外存在的条目（可能为 Forced 字幕）。
   - 除非它明显导致时间轴错乱或与剧情严重冲突。

2. **多余条目需要注意是否为前后错位**
   - 如果发现中文多出来的内容其实是前后某句的错位，应优先判断是否需要重新对齐。

---

## 6. 专有名词统一规则

1. **人物名、地点名、机构名、商品名应前后一致**
   - 特别是跨集字幕任务，例如《Deadloch》第二季等，必须保持统一。

2. **已有固定译名优先**
   - 经典电影、知名人物、官方译名、常见译名应优先采用已有译法。

3. **首次确定后保持一致**
   - 一旦确定某个专有名词译法，后文不得随意变更。

---

## 7. 字幕表达规则

1. **翻译字幕应简洁自然**
   - 避免过度直译。
   - 避免太长、太书面化、太解释性。

2. **符合口语场景**
   - 对白应像人物真实说话，而不是像书面翻译。

3. **保留必要语气**
   - 如讽刺、愤怒、惊讶、调侃、紧张等，应尽量体现。

4. **不滥加解释**
   - 字幕不是注释，不应随意添加原文没有的信息。

---

## 8. 文件输出规则

1. **最终输出和原始字幕相同格式文件，默认为 `.srt`**
   - 例如：
     - `chs.fixed.srt`

2. **保持字幕文件（SRT 等）基本格式**
   - 序号
   - 时间轴
   - 字幕文本
   - 空行

3. **不得破坏时间轴格式**
   - 格式应为：

     ```srt
     00:00:01,000 --> 00:00:03,000
     ```

4. **最终需要总结问题**
   - 完成后总结：
     - 主要错译
     - 主要漏译
     - 重要修正
   - 最好用表格输出。

---

## 9. 最终检查重点

| 检查项 | 要求 |
| --- | --- |
| 时间轴 | 以原始字幕为基准，逐条对齐 |
| 漏译 | 普通对白漏译需补充 |
| SDH | 缺译可忽略 |
| 歌词 | 需要正常翻译 |
| 错译 | 明显错译必须修正 |
| 多余中文条目 | 原则上保持不变 |
| 专有名词 | 前后一致 |
| 风格 | 符合影片背景和人物语气 |
| 文件格式 | 保持标准 SRT 格式 |
| 终稿复核 | 检查错位、漏译、异常长度比例 |
